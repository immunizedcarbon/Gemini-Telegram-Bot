# Neues Projekt – Clean Starter

Dieses Repository wurde als **sauberer Startpunkt** neu vorbereitet.

## Enthalten
- grundlegende Projektstruktur für Python (`src/`, `tests/`)
- minimales `pyproject.toml` für Paket-Metadaten
- `.editorconfig` für konsistente Formatierung
- `.env.example` als Vorlage für Umgebungsvariablen
- aktualisierte `.gitignore`

## Schnellstart
```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Nächste Schritte
1. Paketname in `pyproject.toml` anpassen.
2. Eigene Module in `src/app/` anlegen.
3. Tests in `tests/` ergänzen.
4. CI/CD und Deployment passend zum neuen Projekt ergänzen.
